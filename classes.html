Classes at CCM

This page talks about each class CSIP students took from 2018-2020, which has changed for other generations of CSIP students. Most classes during the first year are IT related, but were more cyber security focused in the second year.

Fall 2018 Semester


  • Intro to IT (CMP 130)
      In Intro to IT, we were taught the basics of IT, from BIOS/UEFI to the internals of a hard drive. We took apart and put back together computers in the lab, and we were even able to fix some that were broken. Intro to IT was a very hands-on class and taught us how to properly handle a computer.

  • Operating Systems (CMP 246)
      Operating Systems was much more than learning about the surface of Windows. In OS, we were taught about Windows, MacOS, Linux, and Unix. With hands-on labs using a virtual machine, we learned about command line, OS functions, and much more.

  • Computer Science I (CMP 128)
      In Computer Science I, or what we called "Java", we learned the basics of the programming language Java, hence the nickname. We learned how to use basic Java packages and libraries, such as java.util, java.lang, and javax.swing. By writing programs for calculators, games, and much more, we were able to write a Java program of our own by the end of the semester.

Winterim 2018-2019


  • Foundations of Info Security (CMP 120)
      In Foundations of Info Security, or "Security" for short, we learned about methods of cyber attacks, encryption methods, and much more on the basics of cybersecurity. We learned to identify a type of attack, and leanred how to use ans efficient method to counter it.

Spring 2019 Semester


  • Visual Basic (CMP 237)
      Visual Basic, also known as VB or VB.net, uses Microsoft's Visual Studio and a drag-and-drop API to develop applications for Windows. We learned how to make basic Windows applications as well as web applications, and got an introduction to graphical design when it came to user interfaces.

  • The Internet & Web Page Design (CMP 239)
      Web Design is focused around, well, designing webpages. We learned how to use HTML and CSS, as well as things we should keep in mind when making a website. This project is actually from this class, and the skills we learned from it are applied to our individual websites.

  • Computers and Data Networks (TEL 107)
      Well known as "Networking", in this class, we learned the basics behind network structures and data transfer. The topics build upon each other as the class goes, and then you suddenly understand why your internet is so bad...

Fall 2019 Semester


  • Routing (CISCO) (TEL 110)
      In routing, we were taught how networks communicate with one another via different mediums, devices, and more. We learned how to use CISCO IOS and Packet Tracer, as well as calculate binary and its meaning in a network and submask.

  • Network Security (CMP 124)
      Network security was one step up from Foundations of Info Security. We learned about threat actors, attack methods, responses to attacks, and essentials in cyber security. It helped prepare us for future classes that we would take later in the year and in college, as some students decided to study cyber security or related fields.

  • Linux (CMP 255)
      Linux is exactly what it sounds like, but much more. At first, we learned the basics of Linux, how to use it, who made it, when it was made, and about different distributions of it. As the semester progressed, however, we began learning about scripting and began writing scripts of our own, allowing automated control of a Linux system.

Spring 2020 Semester


  • Ethical Hacking & System Defense (CMP 243)
      Ethical hacking was a very different class compared to the other ones. Understanding attacks, vulnerabilities, and methods to counteract attacks were almost everyday practices within the class. We did a lot of hands-on labs and simulations allowing us to experience cyber security first hand.

  • Software Engineering (CMP 280)
      Software engineering, though not exactly cyber security, was an important class for many students in the CSIP program, as many are going into the computer science and engineering fields. Learning methods and procedures to approaching a problem and developing solutions as a team is essential for not only us CSIP students, but also as a life skill in many work fields.

  • Info Security Management (CMP 125)
      Info Security Management or as we called it "Info Sec" taught procedures, methods, and other guidelines that companies and organizations follow or need to follow in order to achieve a secure system. Without these procedures, companies are susceptible to cyber attacks and loss of assets.
